# Hangboard Timer

## Description
A web based application for doing hangboard interval training

## Features
* Large display counting down between rest and hang times
* Ability to specify number of sets
* Ability to specify time to hang in seconds
* Ability to specify time to rest in seconds
* Plays a sound when switching between hang and rest
* Section at the bottom for an ad
* Section for an affiliate link to my own hangboard on amazon
* Start/Pause button
* Accessible in terms of focus indicators and button labeling
* Buttons are larger than necessary so they are easy to tap
* All javascript and css and images are in one file for faster downloads.
* Responsive for phone displays
* Save user settings to LocalStorage
* Option to disable audio
